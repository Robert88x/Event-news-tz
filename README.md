- The project is about the event planner.

What the app about;
- user should be able to post event such as birthdays, weddings and different ceremony.
- user should be able to register according country, city, resident and share location to the closed ones and 
  if other user accept the invitation should be able to send notification the day of occurred event
- user should be able to like, comment and share the event posted.
- user should have privillege of post, edit, update and delete the event.
